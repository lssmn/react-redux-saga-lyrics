# react-redux-saga-lyrics

[![Build Status](https://travis-ci.org/lssmn/react-redux-saga-lyrics.svg?branch=master)](https://travis-ci.org/lssmn/react-redux-saga-lyrics)
[![semantic-release](https://img.shields.io/badge/%20%20%F0%9F%93%A6%F0%9F%9A%80-semantic--release-e10079.svg)](https://github.com/semantic-release/semantic-release)

A small lyrics search app built with React and Redux Saga using [create-react-app](https://github.com/facebook/create-react-app).

## Getting started

```sh
$ git clone https://github.com/lssmn/react-redux-saga-lyrics.git
$ cd react-redux-saga-lyrics
$ yarn install
$ yarn start
```

## Tests & Coverage

```sh
$ yarn test
$ yarn test --coverage
```
