# react-redux-saga-lyrics

A small lyrics search app built with React and Redux Saga using [create-react-app](https://github.com/facebook/create-react-app).

## Getting started

To run the app locally, check the following steps:

```sh
$ git clone [url]
$ cd react-redux-saga-lyrics
$ yarn install
$ yarn start
```

## Tests & Coverage

```sh
$ yarn test
$ yarn test --coverage
```
